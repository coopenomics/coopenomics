`trace_api_util` — утилита командной строки (CLI), с помощью которой операторы узла выполняют низкоуровневые операции, связанные с [плагином Trace API](../01_nodeos/03_plugins/trace_api_plugin/index.md). `trace_api_util` поддерживает одну из следующих операций:

* Сжатие файла трассировки `log` в сжатый формат `clog`.

## Использование
```sh
trace_api_util <options> command ...
```

## Опции
Опция (=по умолчанию) | Описание
-|-
`-h [ --help` ] | показать справку

## Команды
Команда | Описание
-|-
`compress` | сжать файл трассировки в формат `clog`

### compress
Сжимает файл трассировки `log` в формат `clog`. По умолчанию имя сжатого файла совпадает с `input-path`, но расширение меняется на `clog`.

#### Использование
```sh
trace_api_util compress <options> input-path [output-path]
```

#### Позиционные аргументы
Опция (=по умолчанию) | Описание
-|-
`input-path` | путь к сжимаемому файлу
`output-path` | [Необязательно] путь к выходному файлу или каталогу

#### Опции
Опция (=по умолчанию) | Описание
-|-
`-h [ --help ]` | показать справку
`-s [ --seek-point-stride ] arg (=512)` | число байт между точками поиска в сжатой трассировке; меньший шаг может ухудшить сжатие, но ускорить чтение

## Замечания
При запуске `trace_api_util` утилита пытается выполнить указанную операцию, возможны исходы:
* при успехе операция выполняется и процесс завершается с кодом 0;
* при ошибке сообщение выводится в `stderr`, код завершения ненулевой.
