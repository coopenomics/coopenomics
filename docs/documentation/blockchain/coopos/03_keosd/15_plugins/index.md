## Обзор

Плагины расширяют базовую функциональность `keosd`. Описание конкретных плагинов — в списке ниже:

* [`http_plugin`](../../01_nodeos/03_plugins/http_plugin/index.md)
* [`wallet_api_plugin`](wallet_api_plugin/index.md)
* [`wallet_plugin`](wallet_plugin/index.md)

!!! note "Модульность keosd"
    Плагины добавляют возможности по шагам. В отличие от подключаемых на лету модулей, плагины `keosd` собираются на этапе компиляции.
