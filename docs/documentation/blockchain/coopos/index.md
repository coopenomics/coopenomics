# Программы узла и справка по командам

Здесь — справка по **COOPOS**: установка, **nodeos**, **cleos**, **keosd**, плагины, HTTP API и утилита `leap-util`. Страницы переводятся на русский; фрагменты вывода консоли и имена подкоманд **cleos** сохраняются как в CLI.

## Состав стека

Тот же набор в концептуальном оглавлении блокчейна: [Программы (инструменты COOPOS)](../programs.md).

- **[Установка](00_install/index.md)** — сборка из исходников, поддерживаемые ОС, вспомогательные средства.
- **[nodeos](01_nodeos/index.md)** — узел сети: запуск, плагины, реплеи, RPC, логирование, концепции, устранение неполадок.
- **[cleos](02_cleos/index.md)** — работа с HTTP API узла и с **keosd**.
- **[keosd](03_keosd/index.md)** — хранение ключей и подпись транзакций.
- **[Утилиты](10_utilities/index.md)** — `leap-util` (журнал блоков, снимки, состояние цепи).

В оглавлении эти материалы собраны под **cleos**: [Ресурсы сети, регистрация и контракты](../system-resources.md); примеры **powerup** — в [руководстве cleos (how-to)](02_cleos/02_how-to-guides/index.md#cleos-howto-powerup). Файл [resources-powerup-ru.md](resources-powerup-ru.md) оставлен как короткая отсылка на тот же раздел.

Остальные страницы дерева доступны по внутренним ссылкам и собираются сайтом документации.
