Документация по основным командам `cleos`

## Команды
- [version](version) — сведения о версии
- [create](create) — создать различные сущности вне и в блокчейне
- [convert](convert) — упаковка и распаковка транзакций
- [get](get) — получить данные и сведения из блокчейна
- [set](set) — задать или обновить состояние блокчейна
- [transfer](transfer.md) — перевести токены с одного аккаунта на другой
- [net](net) — работа с локальными p2p-подключениями
- [wallet](wallet) — работа с локальным кошельком
- [sign](sign.md) — подписать транзакцию
- [push](push) — отправить произвольные транзакции в блокчейн
- [multisig](multisig) — команды контракта multisig
