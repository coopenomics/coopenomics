## Описание
Работа с локальными p2p-подключениями.

## Команда

```console
Usage: cleos net SUBCOMMAND
```

## Подкоманды

 - [cleos net connect](connect.md) — установить новое подключение к пиру

 - [cleos net disconnect](disconnect.md) — закрыть существующее подключение

 - [cleos net status](status.md) — статус существующего подключения

 - [cleos net peers](peers.md) — статус всех пиров
