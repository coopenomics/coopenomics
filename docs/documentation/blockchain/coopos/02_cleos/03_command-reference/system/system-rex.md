## Описание

Действия, связанные с REX (биржа ресурсов).

## Подкоманды

- [deposit](system-rex-deposit.md) — внести в REX-фонд владельца, переводом с ликвидного баланса токенов
- [withdraw](system-rex-withdraw.md) — вывести из REX-фонда владельца на ликвидный баланс токенов
- [buyrex](system-rex-buyrex.md) — купить REX за токены из REX-фонда владельца
- [lendrex](system-rex-lendrex.md) — внести токены в REX-фонд и купить на них REX
- [unstaketorex](system-rex-unstaketorex.md) — купить REX за застейканные токены
- [sellrex](system-rex-sellrex.md) — продать токены REX
- [cancelrexorder](system-rex-cancelrexorder.md) — отменить отложенный ордер на продажу REX, если он есть
- [mvtosavings](system-rex-mvtosavings.md) — переместить токены REX в сберегательный бакет
- [mvfromsavings](system-rex-mvfromsavings.md) — переместить токены REX из сберегательного бакета
- [rentcpu](system-rex-rentcpu.md) — арендовать CPU на 30 дней
- [rentnet](system-rex-rentnet.md) — арендовать сетевую пропускную способность на 30 дней
- [fundcpuloan](system-rex-fundcpuloan.md) — пополнить фонд CPU-кредита
- [fundnetloan](system-rex-fundnetloan.md) — пополнить фонд сетевого кредита
- [defundcpuloan](system-rex-defundcpuloan.md) — вывести из фонда CPU-кредита
- [defundnetloan](system-rex-defundnetloan.md) — вывести из фонда сетевого кредита
- [consolidate](system-rex-consolidate.md) — объединить бакеты погашения REX в один со сроком погашения 4 дня
- [updaterex](system-rex-updaterex.md) — обновить стейк голоса владельца REX и вес голоса
- [rexexec](system-rex-rexexec.md) — выполнить обслуживание REX: обработать истёкшие кредиты и неисполненные ордера на продажу
- [closerex](system-rex-closerex.md) — удалить неиспользуемые записи пользовательских таблиц, связанных с REX
