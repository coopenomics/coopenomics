## Описание

Отправить действие контракта eosio.system в блокчейн.

## Подкоманды

- [system newaccount](system-newaccount.md) — создать новую учётную запись в блокчейне с начальными ресурсами
- [system regproducer](system-regproducer.md) — зарегистрировать нового продюсера
- [system unregprod](system-unregprod.md) — снять регистрацию существующего продюсера
- [system voteproducer](system-voteproducer.md) — голосовать за продюсера
- [system listproducers](system-listproducers.md) — список продюсеров
- [system delegatebw](system-delegatebw.md) — делегировать пропускную способность
- [system undelegatebw](system-undelegatebw.md) — отменить делегирование пропускной способности
- [system listbw](system-listbw.md) — список делегированной пропускной способности
- [system bidname](system-bidname.md) — торги за имена
- [system bidnameinfo](system-bidnameinfo.md) — сведения о торгах за имя
- [system buyram](system-buyram.md) — купить RAM
- [system sellram](system-sellram.md) — продать RAM
- [system claimrewards](system-claimrewards.md) — получить награды продюсера
- [system regproxy](system-regproxy.md) — зарегистрировать учётную запись как прокси (для голосования)
- [system unregproxy](system-unregproxy.md) — снять регистрацию учётной записи как прокси (для голосования)
- [system canceldelay](system-canceldelay.md) — отменить отложенную транзакцию
- [system rex](system-rex.md) — действия, связанные с REX (биржа ресурсов)
