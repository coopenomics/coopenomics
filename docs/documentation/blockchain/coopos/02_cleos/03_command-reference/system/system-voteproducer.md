## Подкоманды

- [voteproducer proxy](system-voteproducer-proxy.md) — голосовать стейком через прокси
- [voteproducer prods](system-voteproducer-prods.md) — голосовать за одного или нескольких продюсеров
- [voteproducer approve](system-voteproducer-approve.md) — добавить одного продюсера в список голосования
- [voteproducer unapprove](system-voteproducer-unapprove.md) — убрать одного продюсера из списка голосования
