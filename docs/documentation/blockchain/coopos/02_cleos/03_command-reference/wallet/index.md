## Описание
Подкоманды для работы с локальным кошельком.

## Подкоманды

- [create](create.md) — создать новый локальный кошелёк
- [create_key](create_key.md) — создать пару ключей в кошельке
- [open](open.md) — открыть существующий кошелёк
- [lock](lock.md) — заблокировать кошелёк
- [lock_all](lock_all.md) — заблокировать все разблокированные кошельки
- [unlock](unlock.md) — разблокировать кошелёк
- [import](import.md) — импортировать приватный ключ в кошелёк
- [list](list.md) — список открытых кошельков, * = разблокирован
- [keys](keys.md) — список приватных ключей из всех разблокированных кошельков в формате WIF
- [private_keys](private_keys.md) — список публичных ключей из всех разблокированных кошельков
