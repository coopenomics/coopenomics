## Описание

Получить текущую информацию о блокчейне

## Позиционные параметры
Параметры не принимаются. 
## Опции
- `-h`, `--help` — вывести справку и выйти
## Пример


```sh
cleos get info
```

Команда возвращает текущее состояние блокчейна.
