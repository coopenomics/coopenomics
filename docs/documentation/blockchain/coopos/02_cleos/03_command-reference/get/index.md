## Описание

Получить различные данные и сведения из блокчейна

## Подкоманды
- [info](info.md) — текущая информация о блокчейне
- [block](block.md) — полный блок из блокчейна
- [account](account.md) — аккаунт из блокчейна
- [code](code.md) — код и ABI аккаунта
- [abi](abi.md) — ABI аккаунта
- [table](table.md) — содержимое таблицы БД
- [scope](scope.md) — список scope и таблиц контракта
- [currency](currency.md) — сведения о стандартных валютах
- [accounts](accounts.md) — аккаунты, связанные с публичным ключом
- [servants](servants.md) — аккаунты-подчинённые (servants) для заданного аккаунта
- [transaction](transaction.md) — транзакция из блокчейна
- [actions](actions.md) — действия, где в авторизации или у получателя указан заданный аккаунт
- [schedule](schedule.md) — расписание продюсеров
- [transaction_id](transaction_id.md) — id транзакции по объекту транзакции
