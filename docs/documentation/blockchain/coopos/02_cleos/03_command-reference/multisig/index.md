## Описание
Команды контракта multisig

## Подкоманды
- [multisig propose](multisig-propose.md) — предложить транзакцию
- [multisig review](multisig-review.md) — просмотреть транзакцию
- [multisig approve](multisig-approve.md) — одобрить предложенную транзакцию
- [multisig unapprove](multisig-unapprove.md) — снять одобрение с предложенной транзакции

- [multisig invalidate](multisig-invalidate.md) — аннулировать все одобрения multisig аккаунта
- [multisig cancel](multisig-cancel.md) — отменить предложенную транзакцию
- [multisig exec](multisig-exec.md) — выполнить предложенную транзакцию
