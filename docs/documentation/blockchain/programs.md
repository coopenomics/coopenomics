# Программы (инструменты COOPOS)

Здесь кратко зафиксирован **состав стека**, который дальше в оглавлении разобран как **«Программы узла»** — те же исполняемые компоненты и темы, что в [Программы узла и справка по командам](coopos/index.md). Цель страницы — связать концептуальные разделы ([Концептуальная модель COOPOS](system-model.md)) с конкретными бинарниками и руководствами.

## Назначение компонентов

| Программа | Роль |
|-----------|------|
| **nodeos** | Узел сети: P2P, применение блоков, исполнение WASM, состояние в памяти, HTTP API через плагины. |
| **cleos** | CLI к HTTP API **nodeos** и к **keosd**: транзакции, запросы к цепи, разработческие операции. |
| **keosd** | Процесс-кошелёк: хранение ключей (в т.ч. зашифрованно) и подпись без раскрытия секрета приложению. |
| **CDT** | Компилятор и инструменты для сборки контрактов в WASM и генерации ABI (разрабатывается отдельно от дистрибутива узла). |
| **leap-util** | Утилиты обслуживания: журнал блоков, снимки, сведения о цепи — см. [Утилиты](coopos/10_utilities/index.md). |

Полное дерево страниц (установка, `config.ini`, плагины, реплеи, RPC, логи): **раздел оглавления «Программы узла»** и корневая страница [coopos/index.md](coopos/index.md).
