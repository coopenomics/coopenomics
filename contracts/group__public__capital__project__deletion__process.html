Контракт CAPITAL » Процессы

Процесс полного удаления проекта и всех связанных данных из блокчейна в случаях, когда хранение данных более не целесообразно.

Назначение процесса

Удаление проекта обеспечивает:

  • Освобождение ресурсов блокчейна от неактуальных данных
  • Удаление проектов, не предполагавших получение членских взносов
  • Очистку системы от завершенных экспериментальных проектов
  • Оптимизацию использования оперативной памяти блокчейна
Предусловие
  • Все вклады участников должны быть сконвертированы
  • Нет неразрешенных финансовых обязательств
  • Проект изначально не предполагал получение членских взносов

Пошаговое описание процесса

Шаг 1: Инициация удаления проекта

Действие: Удаление проекта

Кооператив инициирует процесс удаления проекта после завершения всех финансовых операций и конвертации вкладов участников.

Постусловие
Проверка авторизации кооператива

Шаг 2: Проверка статуса проекта

Действие: Валидация состояния

Система проверяет, что проект находится в корректном статусе для удаления. В текущей реализации требуется статус VOTING, но в продакшене логично требовать CLOSED статус.

Постусловие
Подтвержден корректный статус проекта

Шаг 3: Валидация конвертации вкладов

Действие: Проверка завершенности финансовых операций

Перед удалением система должна проверить, что:

  • Все сегменты участников сконвертированы
  • Нет неразрешенных финансовых обязательств
  • Все средства переведены в целевые кошельки или проекты
Постусловие
Проверка завершенности всех финансовых операций

Шаг 4: Удаление связанных данных

Действие: Очистка всех таблиц проекта

Система должна удалить все данные, связанные с проектом:

  • Записи участников и их сегменты
  • Данные голосования и результаты
  • Инвестиции и финансовые операции
  • Коммиты и трудозатраты
  • Расходы и ссуды
Постусловие
Удаление всех связанных записей из таблиц блокчейна

Шаг 5: Удаление записи проекта

Действие: Финальная очистка

После удаления всех связанных данных система удаляет основную запись проекта из таблицы проектов.

Постусловие
Полное удаление проекта из блокчейна
Освобождение ресурсов оперативной памяти
Предупреждения
Текущая реализация не завершена и находится в разработке.
