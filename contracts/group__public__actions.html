Подробное описание

Методы действий контрактов

Действия представляют собой методы, которые можно вызывать в контрактах. Они отвечают за выполнение определенных задач, таких как создание аккаунта, обновление информации или осуществление транзакции. Эта группа описывает все доступные действия, которые предоставляют контракты кооперативной экономики.

Заметки

Диаграмма процесса и inline транзакций:

  1. registrator::adduser (
    • добавляем аккаунт
    1. soviet::adduser(
      • добавляем пайщика
      • добавляем кошелёк с минимальным взносом
      1. gateway::adduser(
      • устанавливаем дату вступления
      • фиксируем принятый взнос в реестре взносов
        1. fund::addcirculate (добавляем минимальный взнос)
        2. fund::spreadamount? (опционально распределяем вступительный взнос по фондам) ) ) )
