    Реестр документов (registry_id)¶

    Соответствие номера в реестре, человекочитаемого названия и файла шаблона в репозитории фабрики. Логика генерации для каждого типа — в парном файле в каталоге Actions с тем же префиксом номера.

    Базовый путь к шаблонам: components/factory/src/Templates.
